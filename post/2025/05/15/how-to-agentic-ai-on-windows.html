Building Agentic AI with LangChain: A Beginner's Guide for Windows 10/11

This step-by-step tutorial will teach you how to create AI agents using LangChain on Windows 10 or 11. Even if you're a complete beginner, you'll be able to build an AI that can think, make decisions, and take actions based on your instructions.

What Are Agentic AI Systems?

Agentic AI systems are artificial intelligence applications that can:

  • Make decisions about what actions to take
  • Use tools to accomplish tasks
  • Interact with various data sources
  • Plan and execute multi-step processes

Unlike simple chatbots that just respond to messages, agents can actively do things for you - like searching the web, analyzing data, or even controlling other software.

Prerequisites

Before we start, make sure you have:

  1. Windows 10 or 11 computer
  2. Python (3.9 or higher) installed
    • Download from python.org
    • During installation, check the box for "Add Python to PATH"
  3. Basic understanding of coding (but don't worry, we'll explain everything!)

Section 1: Setting Up Your Environment

Step 1: Install Python

If you haven't already installed Python:

  1. Go to python.org/downloads
  2. Download the latest version for Windows
  3. Run the installer
  4. IMPORTANT: Check the box that says "Add Python to PATH"
  5. Click "Install Now"

Step 2: Create a Project Folder

  1. Open File Explorer
  2. Create a new folder where you want to store your project (e.g., C:\LangChainAgent)
  3. Open Command Prompt:
    • Press Win + R, type cmd, and press Enter
  4. Navigate to your project folder:
    cd C:\LangChainAgent

Step 3: Set Up a Virtual Environment

Creating a virtual environment helps keep your project's dependencies separate from other Python projects.

  1. In Command Prompt, create a virtual environment:

    python -m venv venv

  2. Activate the virtual environment:

    venv\Scripts\activate

    You should see (venv) appear at the beginning of your command line.

Step 4: Install Required Packages

Now we'll install LangChain and other necessary packages:

pip install langchain langchain-community langchain-openai python-dotenv

This may take a few minutes as it downloads and installs all the required packages.

Section 2: Getting API Keys

LangChain works with various language models. For this tutorial, we'll use OpenAI's models, which require an API key.

Step 1: Get an OpenAI API Key

  1. Go to platform.openai.com
  2. Sign up for an account or sign in
  3. Navigate to API keys section
  4. Create a new secret key
  5. Copy the key (you won't be able to see it again!)

Step 2: Set Up Environment Variables

  1. In your project folder, create a new file called .env
  2. Open the file with a text editor (like Notepad)
  3. Add your API key:
    OPENAI_API_KEY=your_api_key_here
  4. Save and close the file

Section 3: Creating Your First Agent

Now let's build a simple agent that can perform tasks and use tools.

Step 1: Create a Python File

  1. In your project folder, create a new file called simple_agent.py
  2. Open the file with a text editor

Step 2: Write the Basic Agent Code

Copy the following code into your simple_agent.py file:

# Import necessary libraries
import os
from dotenv import load_dotenv
from langchain_openai import ChatOpenAI
from langchain.agents import AgentExecutor, create_openai_functions_agent
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain.tools import DuckDuckGoSearchRun
from langchain.memory import ConversationBufferMemory

# Load environment variables from .env file
load_dotenv()

# Initialize the language model
llm = ChatOpenAI(temperature=0)

# Create a search tool
search_tool = DuckDuckGoSearchRun()
tools = [search_tool]

# Create a prompt template
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful AI assistant with access to tools. Use them when necessary to best assist the user."),
    MessagesPlaceholder(variable_name="chat_history"),
    ("user", "{input}"),
    MessagesPlaceholder(variable_name="agent_scratchpad")
])

# Set up memory for conversation history
memory = ConversationBufferMemory(memory_key="chat_history", return_messages=True)

# Create the agent
agent = create_openai_functions_agent(llm, tools, prompt)

# Create the agent executor
agent_executor = AgentExecutor(
    agent=agent,
    tools=tools,
    memory=memory,
    verbose=True
)

# Main interaction loop
def main():
    print("Welcome to your AI Agent! Type 'exit' to quit.")
    
    while True:
        user_input = input("\nYou: ")
        
        if user_input.lower() == 'exit':
            print("Goodbye!")
            break
        
        response = agent_executor.invoke({"input": user_input})
        print(f"\nAgent: {response['output']}")

if __name__ == "__main__":
    main()

Step 3: Run Your Agent

  1. In Command Prompt (with your virtual environment activated), run:

    python simple_agent.py

  2. Start interacting with your agent by typing questions or tasks!

Section 6: Creating a Custom Tool

Let's create an agent with a completely custom tool:

Step 1: Create a New File

Create a file called custom_tool_agent.py in your project folder.

Step 2: Add a Custom Wikipedia Tool

Copy this code, which adds a tool that summarizes Wikipedia articles:

# Import necessary libraries
import os
import wikipedia
from dotenv import load_dotenv
from langchain_openai import ChatOpenAI
from langchain.agents import AgentExecutor, create_openai_functions_agent
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain.tools import DuckDuckGoSearchRun, tool
from langchain.memory import ConversationBufferMemory

# Load environment variables from .env file
load_dotenv()

# Check if Wikipedia is installed, if not install it
try:
    wikipedia.summary("Python programming")
except:
    import subprocess
    subprocess.check_call(["pip", "install", "wikipedia"])
    import wikipedia

# Initialize the language model
llm = ChatOpenAI(temperature=0)

# Create a search tool
search_tool = DuckDuckGoSearchRun()

# Define a calculator tool
@tool
def calculator(expression: str) -> float:
    """Evaluate a mathematical expression."""
    return eval(expression)

# Define a Wikipedia tool
@tool
def wiki_search(query: str) -> str:
    """Search Wikipedia for information on a topic."""
    try:
        # Search for the page
        page = wikipedia.page(query)
        # Get a summary (first 3 sentences)
        summary = wikipedia.summary(query, sentences=3)
        return f"Title: {page.title}\nSummary: {summary}\nURL: {page.url}"
    except:
        return f"Could not find a Wikipedia page about '{query}'."

# Combine all tools
tools = [search_tool, calculator, wiki_search]

# Create a prompt template
prompt = ChatPromptTemplate.from_messages([
    ("system", """You are a helpful AI assistant with access to tools.
     When asked about mathematical calculations, use the calculator tool.
     When asked for factual information about a specific topic, use the wiki_search tool.
     For other information you don't know, search using the search tool.
     If no tools are needed, just respond directly."""),
    MessagesPlaceholder(variable_name="chat_history"),
    ("user", "{input}"),
    MessagesPlaceholder(variable_name="agent_scratchpad")
])

# Set up memory for conversation history
memory = ConversationBufferMemory(memory_key="chat_history", return_messages=True)

# Create the agent
agent = create_openai_functions_agent(llm, tools, prompt)

# Create the agent executor
agent_executor = AgentExecutor(
    agent=agent,
    tools=tools,
    memory=memory,
    verbose=True
)

# Main interaction loop
def main():
    print("Welcome to your Custom Tool AI Agent! Type 'exit' to quit.")
    print("This agent can search Wikipedia, perform calculations, and search the web.")
    
    while True:
        user_input = input("\nYou: ")
        
        if user_input.lower() == 'exit':
            print("Goodbye!")
            break
        
        response = agent_executor.invoke({"input": user_input})
        print(f"\nAgent: {response['output']}")

if __name__ == "__main__":
    main()

Step 3: Run Your Agent with Custom Tools

  1. In Command Prompt (with your virtual environment activated), run:

    python custom_tool_agent.py

  2. Try asking about specific topics to test the Wikipedia tool:

    • "Tell me about Albert Einstein"
    • "What can you tell me about the Python programming language?"

Examples of What Your Agents Can Do

Example 1: Simple Information Lookup (Easy)

Try asking your agent:

  • "Who is the current president of the United States?"
  • "What is the capital of Japan?"

The agent will use its search tool to find current information.

Example 2: Calculations and Data Processing (Moderate)

Try asking your agent:

  • "If I have 125 items and each costs $3.99, what's the total cost?"
  • "What's the square root of 529?"

The agent will use its calculator tool to compute answers.

Example 3: Complex Multi-step Tasks (Complex)

Try asking your agent:

  • "Find information about climate change, then summarize the main causes and effects"
  • "I need to plan a trip to Paris. What are the top attractions, and what's the weather like there now?"

The agent will combine multiple tools and reasoning steps to provide comprehensive answers.
